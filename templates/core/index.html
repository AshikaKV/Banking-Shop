Simple Banking System

This application is entitled Simple Banking Management System. This is a simple web-based application developed in Python and Django Framework. The main goal of this project is to provide bank account holders can make some transactions. The application allows the account holders of a certain bank to withdraw and deposit. It has a pleasant user interface and this application consists of user-friendly features and functionalities.
